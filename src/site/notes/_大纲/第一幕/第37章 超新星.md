# 第37章 坍缩

阵核的坍塌以一种出乎所有人预料的安静开始了。

最先瓦解的是外围菌毯。失去了阵核那颗暗红心脏持续千年的脉动信号，覆盖整片空域的暗色物质像退潮的海水，一层一层地失去色泽、失去形态、失去构成它们存在的最后一点熵余者意识。菌毯在瓦解过程中发出了一种极其古怪的声音，像无数根被绷到极限的琴弦在同一时刻断裂，但每一根断裂时发出的音高都不一样。那声音叠在一起形成一道从极高滑向极低的、绵延数百里的下行音阶，在时空之海的真空中找不到介质传播，便顺着菌毯自身的暗色纹路向内传导，最后全部汇聚到了阵核中央已空无一人的球形空间中，在那里炸开成一圈无声的、暗绿色的冲击波。

六颗恒星锚点紧随其后。锁链断开的瞬间，它们各自失去了作为阵核能量来源的唯一意义。它们在这一千年里被凤凰王反复释放能量的反灌冲击了无数次，核心早已不稳。如今最后的锚定链接也消失了，它们同时开始坍缩。赤、金、冰蓝、银白、深紫、青绿，六颗恒星在时空之海的六个方向上同时向中心收缩，又在收缩到极限时各自以不同的颜色炸开成六团微型的、安静的、近乎礼貌的超新星遗迹。它们为这个世界的千年劫难提供了最后的能量，然后燃尽了自己。

琉玥载着三人沿银线的反方向全速飞行。菌毯瓦解后留下的真空中漂浮着大量未完全消解的暗色碎片，迫使她不得不在高速中同时维持冰火双翼的平衡，左翼的冰棱不断击碎迎面而来的碎片，右翼的火焰将擦身而过的残余菌毯烧成飞灰。云涡紧贴在琉玥左翼侧后方，光带触须全部绷直如弦，将白月霖牢牢固定在平台上。白月霖跪坐在云涡表面，双手还保持着方才拉锁链的姿态，手指微微蜷着，指节泛白，掌心残留着锁链碎屑融化后渗入皮肤的微凉触感。她没有回头去看阵核的坍塌。只是盯着自己掌心里那些正在缓慢融入皮下的金红与银白星尘，看着它们在自己的残月印记上留下一层极淡的、如同旧照片表面的珠光。

"他在锁链断掉之前说了什么没有。除了那些关于孩子的话。"星璃娅的声音从云涡前方传来，语调很平，但每个字都放得比平时慢，她知道白月霖需要时间把那些话从心里翻译到嘴边。

"说了。他说——'你长得像你母亲。岚烜更像你们父亲一点。他每次犯错被我抓到都会先低头，然后从下往上这样看人。就你现在这样。'然后笑了。笑了两次。第一次是被我认出来的时候。第二次是听到枯枕的孩子还活着的时候。"她将手翻过来，看着掌心里最后一点星尘彻底融化，然后抬起头，那双已承载了两位神祇最后遗泽的红瞳在六颗恒星同时坍缩的壮观背景中显得格外安静，"他走的时候，不痛苦。"

星璃娅没有继续问。她将云涡的航速调到最大，同时对琉玥做了一个加速的手势。身后，阵核的球形空间正在从边缘开始向内逐层塌陷，先是六色光幕的残余波纹被卷入中心那团正在迅速缩小的暗红残光，然后是凤凰王千年来释放后又反灌回锚点的所有恒星能量残渣，在阵核中心凝聚成一颗极小的、密度极高的、包含了所有未竟之事的重力奇点。奇点存在了大约三次呼吸的时间。然后它蒸发了。在蒸发之前，它向时空之海的六个方向各投射了一道由纯粹光速构成的、裹挟着两位神祇最后神力余韵的冲击波。冲击波扫过正在崩解的菌毯残骸，扫过六颗坍缩的恒星遗迹，扫过远方那颗赤红与冰蓝各占一半的、孤零零悬在宇宙角落里的弹珠般的小星球，然后消失在无边无际的星海深处。

白月霖感觉到自己胸口的残月印记在冲击波穿过身体时轻轻震了一下。那震动里，她分辨出了一个极其微弱的、几乎淹没在冲击波庞大能量中的信号：祈尔米修罗残存意识的最后一道指令：散逸在时空之海中的所有锚定能量已经完成了回收。从今往后，这个世界的时空结构将由新的锚定神力持有者自行维系。

"他最后给我的那道指令——"白月霖低头看着自己胸口的残月印记，印记的边缘正在以极其缓慢的速度向外延伸出一圈新的纹路，"是把这个世界锚定在我的神力上。"

"嗯。"星璃娅的语气和第一次见面时评价她的烤鱼一样波澜不惊，"新神登基的第一个工作任务：全天候维系一颗行星的时空稳定。没有工资。不能辞职。"

琉玥在高速飞行中仍然顽强地偏头插了一句："主人，我记得你当年登基的时候第一个任务是给七十六颗行星重新校准公转轨道。然后你罢工了。"

"那不一样。那些行星上没有人给我烤松饼。"

白月霖愣了一下，然后噗嗤笑出了声。那笑声在正在崩塌的千年牢笼的余波中显得很小，很轻，很不合时宜。但那笑声在。在就够了。
