# 第1章 你醒啦？

青草葳蕤，在清风吹拂下缓缓摇曳，几株淡蕊点缀其中，在空气中散发着清新的气息。

清晨的第一缕阳光破开雾霭，映衬着湖面泛起鳞光，也穿过驳杂的冷杉树叶，照在少女白皙的脸庞微微发烫。

微风与鸟鸣在耳畔将她呼唤，少女的眉睫微微发颤，惺忪睁开了眼。

她的眸子宛如被一颗囚禁着的炽烈的蓝白恒星，从中静谧延伸出四条神圣光痕，浮衬于缀满细碎星子的宇宙深蓝底色之上。

"我这是，在哪里？"她从草地上坐起，茫然地打量起四周。四下无人，只有未被惊扰的自然景色。

目光掠过湖面，掠过冷杉林，掠过远处天际那轮略显陌生的淡金色太阳——忽然间，一缕若有若无的熟悉感从心底浮起，像水面下倏忽游过的鱼影，还未来得及捕捉便已消散。她皱了皱眉，试图抓住那丝感觉，却只捞到一手空茫。

算了，大概是刚醒来的错觉。

少女扶额，那里还偶尔传出微弱的痛觉，像是在梦里被人做了开颅手术，醒来之后麻醉药效也已经散去。她挪开手，看着自己的手掌，一股奇怪的感觉涌上心头，怎么总感觉……比印象里的要小不少？

目光向皓白的手臂上划去，落在肩膀上，她像是突然想起了什么般，低头朝下看去。

"啊……糟糕"，为什么曾经引以为傲的土地变得如此贫瘠？她喃喃自语，有些悲伤。

是了，她仿佛想起来了什么，咬起食指，露出若有所悟的表情。

是了，她是统领数百位神祇的主神。在争夺最后一片星域的关键一战，她依旧冲在最前方，那时谁也不敢与她对视，璀璨的金色光芒在她背后升起，她的神性光辉笼罩了整片战场，令那些野蛮的、从前只会相互内斗的部落膝盖发软，大批大批地朝她跪拜。

本来以为一统星域的大业就要告成，自己将被奉为无上的主神之首，留下千古芳名供后来者俯首膜拜。然而却不知哪里窜出来一个耀眼的光团，它以一种快到连她的眼里也出现残影的速度飞驰而来，然后……巨大的轰鸣声在她耳边呼啸炸开，那个光团子和她爆了。

还好千钧一发之际，她快速地将自己的神格解体，发散到宇宙各处，自己的主魂则慌忙裹挟着其中的零星几块神格碎片，飞向遥不可知的星球。

看来是最后被人使坏坑了一把，不对，应该是有其他的坏神临门插了一脚，真是太可恶了。少女暗暗咬牙。幕后黑手恐怕正在偷着乐，然后一边忙着接手她已经下好的一盘好棋吧。

她压了压气得起伏不定的胸口，"难怪醒来后整个人都变小了，原来是神格能量丢失的缘故，导致身体倒退回了幼神时期的大小。"

她现在该怎么办？总不能顶着现在这幅模样直接杀回去。以现在这具幼神身躯残留的神力——当然，这个"微弱"是相对于她全盛时期而言，在这偏远星球上足以自保，但若说重返星域夺回王座——别说一统星域了，到那里人家都会问你是哪里来的小不点，一听到自称是最伟大的主神星璃娅，他们就会笑得合不拢嘴，慈祥温柔地跟你说：小妹妹别闹，我带你去找失散多年的爸爸妈妈……

一想到这里，星璃娅就又瘫倒躺回草地里。一想到这个陌生的世界，破碎的关系，就让人陷入一片迷茫。

被忽略了许久的鸟啼再次轻声响起，微风略过草尖，抚摸鼻翼，引起轻微的瘙痒。温煦的日光在树叶沙沙声中摇曳，在少女完美的胴体上留下斑驳的影子。

就当做是一次成长路上的小小挫折，收拾好行装依旧可以自信出发。她需要做的，是在这偏远之地慢慢恢复力量，随后慢慢地朝内星群进发，找寻回那些属于自己的神格碎片。终有一日，她会迎着无上的光辉，以无以匹敌的力量，浴血回到那个独属于她的王位。

"嗯，就酱紫。"少女点点头，再次从草中坐起，心中有了打算之后，她感到信心满满，无比振奋。然而，肚子却不合时宜地紧随其后咕噜叫了一声。

星璃娅这才想起，身上携带的寥寥几片碎片只赋予了她曾经的部分能力。显然，"不需要吃饭"这个能力已经不知道被甩飞到哪片星域了。

她寄希望于已有的碎片，探入精神力检查了下这些碎片的特性。结果大失所望，这些能力似乎自己从没用过，以前都是抛在脑后让它们发霉。在方才的慌乱逃窜之时居然只带来一堆废物，星璃娅摆出一副苦瓜脸想到，难道只能在这破地方自力更生了吗？

她不死心，又反复探查了意识海里面的这几片废物碎片，在不懈努力之下找到了她记不清什么时候留下的一点印记，她也忘了当初为什么留下这个印记，但是事已至此，只能活马当死马医了。她将微弱的神力注入其中。

碎片亮了，发出幽蓝的光芒，从星璃娅的体内飞出，悬浮在少女的面前。

片刻后，那碎片上方投影出一个影像，里面是一位蓝白长发的狐狸少女，月光从天穹撒下，她坐在万丈悬崖边上，画面正对着狐女的侧脸，她百无聊赖地哼着什么，荧光粒子在耷拉着的尾巴和耳尖环绕，翡翠色的眼底闪烁着哀伤落寞的光。

"琉玥！"星璃娅看到这个女孩，就像抓到了救命稻草。那画面中的少女突然惊觉，左右摇头寻察呼声来源，最后她将目光转向星璃娅的一刹那，眼底的落寞突然烟消云散，两只毛茸茸的耳朵"唰"地竖起来。

她幻化成雪狐模样，快速跑了过来，影像被她的那张狐脸塞满，"主人，你在哪里？我想死你啦！"，她在那边欢快地笑着，激动的举措仿佛刚遇到一年未见的老友。

星璃娅的心底闪过一抹不安。看来那个印记是她曾经刻下来用来与自己的宠物琉玥远程联系，因为之后一神一兽经常相伴一起，没有分离，这个手段也被渐渐遗忘，在记忆的角落里沾满了灰。问题是，虽然联系上了可以帮到自己的宠物，但是他们彼此之间也许相隔很远，两者虽能远程见面，却也可能对星璃娅当下的境况爱莫能助。

不管如何，星璃娅都将自己的遭遇告诉了琉玥。

"如果太远，你就不要过来了，我自己想办法。"星璃娅对着她说，谁知琉玥却摆出一副义不容辞的模样，轻哼道："没事的，我看了星图，主人那边离我很近，而且保护主人可是窝滴使命，看我马上就飞过来~"，转瞬她就在影像中一闪而过，消失不见，看起来已经在朝这边进发了。

琉玥这个家伙还是老样子……少女摇了摇头，嘴角扬起轻微的弧度。不出意外，自家宠物身上就携带一个碎片，在当时的混乱之中被同时抛射出去，本来应该会不知所踪，没想到才过不久就遇见了。

虽然她没有说过来需要多久，但以她的性子，恐怕是一路加速，用不了多时就会闪亮登场。在此之前，可以先在旁边的湖里抓点小鱼小虾垫一下肚子。

时间眨眼到了晌午，空气变得闷热，星璃娅在湖边空地升起了火堆，烤炙着刚刚捕获的草鱼。

终于在紧赶慢赶的一阵烧烤之后，星璃娅迫不及待地咬上了一口烤鱼。她细心嚼了嚼，然而味道……只能说充满了田园的气息，而且有点扎嘴，外加外糊里生的口感，不是说不能吃，只能说与她以前享用的那些珍肴美味比起来相差得实在太远。当然，也可能是她从未下过厨的缘故。但是这么说，一介主神竟然沦落到会饿肚子，甚至不得不自己动手做饭，简直是落魄得不能再落魄了。

就在星璃娅思考着怎么把手里的烤鱼消化之时，天空划过一道璀璨的流星，冰白色的尾痕在热烈的日光下也熠熠生辉，流星轰然一声落地。星璃娅知道，她的救星来了。
