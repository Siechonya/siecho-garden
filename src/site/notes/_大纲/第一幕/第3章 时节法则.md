# 第3章 时节法则

不多时，翅尖已掠到小城近郊。琉玥扇动的冰凌羽翼渐缓，尾尖扫过低空的流云，仰头问："主人，咱要从大门进吗？"

星璃娅懂她的顾虑。寻常城池为了治安，多半设着飞行禁令——换作在她的地界，规矩自然管不到她头上，但这陌生地方，贸然飞进去被拦截甚至围攻，倒也麻烦。

不过嘛……星璃娅的嘴角勾出一弯月牙似的笑，眼尾泛着点小兽般的狡黠。她向来随性惯了，哪肯受这约束？何况身边还有琉玥在，有什么好怕的？

"你会保护好我吧？"她轻咳一声清了嗓，刻意拉高了声线，尾音缀着点软糯。

"当然啦！玥玥要一辈子守着主人！"琉玥头点得像捣蒜，鼻尖绷得紧紧的，模样认真得很，话音未落，便振翅加速，直直扎向城池上空。

小城边界果然浮着层透明的能量薄膜。膜外还是阳光微醺、绿茵铺地的暖景，穿过后却骤入另一片天地——天穹上正簌簌落着小雪，积雪漫过红砖绿瓦，淌满街头巷尾。

钟楼尖顶戳破飘雪的云层，十二道青铜铃舌冻在风里，早凝了半尺冰棱。星璃娅的目光被穹顶边缘垂落的冰锥勾住——每根晶锥里都冻着片不一样的叶子，或是银杏，或是梧桐，此刻嵌着落叶松针叶的那支，正透过冰面散着淡银的星芒。这大抵是小城的季候表——落叶松该是代表微寒，等轮转到梧桐叶时，穹顶外或许就该飘梧桐雨了。

这般精巧的装置，倒像是神祇的手笔。星璃娅便让琉玥载着往钟楼去，雪狐硕大的影子投在雪地上，地上的人纷纷仰头，指着她们窃窃议论。

"那是什么？竟然能在天上飞，还是只雪狐狸？"有人揉着眼睛惊诧，"这狐狸的冰凌羽毛真亮，跟院长大人的鸦羽似的。"

星璃娅在天上还不知情——这地方哪有什么飞行禁令？穷乡僻壤的，能御空的人本就凤毛麟角。

快到钟塔正面时，光线被钟楼阴影吞掉的地方，立着个人影。那人裹着一袭黑袍，衣料像浸在无尽夜色里的暗流，沉得发哑，带着些历史磨出的沉默威严。宽大的袍袖仿佛能兜住风与月，袖口织着细密的银线暗纹，在阴影里蜷着，像沉眠的灵蛇。黑袍兜帽下，面容若隐若现，带着点胡茬的俊朗脸上挂着礼貌的笑，目光像能穿透雾，直直落在星璃娅身上。

"两位小姐。"她们刚落地，男人便抬手按在胸前，长指扣着衣襟微微躬身，"欢迎来到雪的国度，我在此已等候多时。"

刚幻成人形的小雪狐懵了，左看右看，还以为他在等别人。

星璃娅轻咳一声，换上副端庄腔调："多谢。阁下在此等候，可是专为我们？"

黑袍男人直起身，目光里带着丝探寻："正是。我叫黎敖，诸位可当我是这片偏远地界的守护者。方才感知到异常气息靠近，料想是远方贵客，便在此恭候。"

琉玥这才反应过来，立刻警惕地挡在星璃娅身前，瞪着眼问："你不会有什么坏心思吧？"

黎敖轻笑一声，摊开双手："二位放心，我并无恶意。这雪国许久未有外人到访，我不过是好奇二位来历，再者也想尽尽地主之谊。"

星璃娅拍了拍琉玥的肩膀："既然如此，便多谢黎先生好意。我们到这里，只是对那穹顶上的冰锥好奇，想研究一二。"

她心里却在嘀咕：他肯定不会轻易交这种宝贝，不过……实在不行就自己拿呗——怎么忽然有点想干坏事的雀跃？嗯，万一他不乐意，就先绑起来，再从他嘴里撬点消息。

"小姐若喜欢，我为你摘下便是。"黎敖竟笑着应了。

星璃娅愣了愣——这么痛快就答应了？

黎敖心里也在犯嘀咕。按他预想，本该是二人先亮身份，再求他介绍国度，他顺势宴请，事后谈合作——哪想到刚见面，这人就直愣愣要白拿他家东西？而且她看自己的眼神怎么怪怪的，怎么后脊忽然有点发寒？略一斟酌，他决定先顺着对方的心意，看看后续再说。

只见黎敖脚底忽生青风，人影一闪便掠到空中，再落地时，手里已捏着支黯淡的冰锥。

"二位可别模仿我的动作，否则恐遭神谕惩戒。"他伸手递出冰锥，银灰色的冰锥表面簌簌掉着细碎冰尘，落地便化了。待冰壳褪尽，一颗赤红晶珠浮在他掌心，半截柳絮在珠里散着微弱的荧光。

星璃娅伸手接过，晶珠圆润的弧面贴在掌心，触感微微发热。琉玥也凑过脑袋，好奇地盯着看。

"小姐可满意？"黎敖微笑着问。这半截柳絮是"春意"的法则，早已许久未启用——不是他不愿拿更好的，实在是这小城全靠冰雪法则维系，只能把这些快荒废的法则让出来。

星璃娅掂了掂手里的晶珠，心里撇嘴：这家伙真抠搜，里面的神力都快散尽了。她指尖悄悄渗进丝神力，却意外发现了些端倪——这晶珠的神力溯源分了两道，其中一道，竟正好落在城内。

另一道，则指向城外极远处——但那方向的波动更加微弱，像是被什么刻意隔断了。

她没有声张，只是将晶珠收入袖中。

"多谢。"星璃娅收了晶珠，忽然转了话头，"请问你们这里有星槎吗？"

"星……星什么？槎？星什么槎？"黎敖皱着眉，显然没听过。

"就是种远距离航行的工具，能在时空之海飞的。你们这里没有类似的东西？"

"小姐说笑了，在下从未见过。"黎敖这辈子都没离开过这颗星球，更别提星际航行了。但这几句话，倒让他更确定眼前人不一般——或许是个机会？他连忙补充，"若是小姐有法子造星槎，我们可以提供材料。"

"造船是要炸星球的哦。"一旁的小雪狐忽然冒出来，晃着尾巴解释，"星槎要在时空之海飞，得用巨多能源——造一艘，消耗的能量得按几颗星球算呢。"

黎敖脸上的笑瞬间僵住，身体下意识地往后倾了倾，像是想躲远些。他喉结微微滚动——这个下意识的小动作，让那一身精心维持的优雅出现了第一丝细小的裂纹。

星璃娅注意到了，唇角微微一勾。原来这位"守护者"大人在涉及自身安危的事上，并不像外表看起来那么沉稳嘛。

她叹了口气——果然不该对这种小地方抱期望。看来短时间内是离不开这颗星球了。想走，要么把这颗星球的能量吸光——瞧这光景，这点能量怕是不够；要么就得找到这世界的古老神祇，借点神力来用。

她从前总借着一身强韧神力在宇宙里横冲直撞，这会儿倒忘了，自家这小狐狸，本就是现成的"坐骑"。
