# 第32章 深入

离开神殿废墟之后，菌毯的厚度开始以令人不安的速度增长。

原先只是覆盖在空域表面的薄层，如今已堆积成一座座暗色的、半流质的、以极其缓慢的频率起伏着的丘陵。它们彼此挤压、重叠、沿着银线两侧绵延展开，像某种巨型生物休眠中的消化管道横截面。银线本身也在变——从废墟中延伸出来时不过是一缕勉强可辨的微光，越往阵核方向深入，就越发明亮、越发稳定，到后来已如一条被冻在冰层最深处的闪电，固执地劈开前方每一寸黑暗。菌毯避开了银线本身，却密密匝匝地挤在银线两侧，形成一道仅容二人并行的狭长甬道。甬道的壁面湿漉漉地泛着暗绿色的荧光，每隔数息便会集体脉动一次，脉动的方向与阵核那颗暗红心脏的收缩节律完全同步，仿佛整片菌毯都是那颗心脏延伸出来的循环系统。

琉玥飞在最前面。她的冰棱羽翼在甬道中被迫收拢了大半，翼展缩减到只剩原来的三分之一，即便如此，翼尖的极光粒子偶尔擦过菌毯壁面，还是会激起一片短促而尖锐的嘶嘶声。那些被擦过的暗色物质会在瞬间剧烈扭动，像是在沸油里滴入了冷水，随即迅速恢复原状，只留下几缕被烧焦后缓缓飘散的青烟。琉玥每听到一次这种嘶嘶声，左耳就会往外转五度，又迅速弹回来。

星璃娅立在云涡之上，与琉玥保持三丈距离，掌心托着一团经遗辉结晶放大的锚定频谱扫描信号。她每隔十息将扫描脉冲向前方发射一次，每一次反馈回来的图像都比上一次更清晰一分：菌毯的厚度在前方某个位置达到了峰值，然后骤降，骤降到几乎为零。那里有一个空洞。一个被菌毯层层包裹却拒绝被侵入的、约莫百丈直径的球形空间。空间的中心悬浮着一样东西。扫描信号打在上面，反馈回来的既非固体也非能量场，而是锁链。

准确地说，是六条由纯粹星光凝成的锁链。每一条都从六芒星的一个锚点延伸而来，直径足以缠绕一颗小型卫星，却在靠近那个球形空间的中心时急剧收缩，收束为一个只够锁住一人身躯的环。锁链的末端隐没在一团浓稠的金红色光芒之中，那光的质地既像火焰又像熔岩，但在星璃娅的感知频谱里，它呈现出一种与熵余者菌毯截然相反的痛苦：菌毯的低语是冷的、狡猾的、像冰水沿着脊椎往下渗。那团光的低语是烫的、破碎的、像一个人在着了火的房间里反复念着同一个名字，念到声带撕裂也不肯停。

白月霖伏在琉玥背上，一直在低声数数。那并非紧张，而是在维持锚定脉冲的节奏：咚、咚、咚。每隔百息一次，她已经坚持了不知多少个百息，嘴唇因为反复默念同一个数字序列而微微发白。琉玥偏头用鼻尖碰了碰她的膝盖，冰凉的触感让她从那种近乎出神的状态中短暂脱离。

"快到了。我感知到阵核外围的结界波动了。"琉玥的声音里那层紧绷的壳下透出一点极力压制的兴奋，随即狐耳往后转了两圈，听见了什么东西。星璃娅也听见了。是菌毯安静下来了。

那些在甬道壁面上按照心脏节律脉动了不知多少年的暗色纹路，在她们距离阵核只剩最后一小段银线的时候，集体停止了脉动。整片菌毯陷入了一种比死亡更令人不安的绝对静止，像是某种蛰伏在暗处的庞然大物忽然屏住了呼吸。千万条暗色纹路的末端在同一时刻微微抬起，朝向了甬道中央的三个入侵者。没有攻击。没有低语。只是看着。

"它们在等。"白月霖的声音很轻，但在这片忽然安静下来的菌毯甬道中清晰得过分，"等我们走到银线的尽头——阵核的入口。那里应该有一道屏障。它们想看看我们能不能打开它。"她低头看了一眼自己的手。锚定脉冲还在，幽蓝光晕稳定地裹着每一根手指。她对着掌心轻轻吹了口气，光晕漾开一圈涟漪随即重新合拢。

"走吧。有人在屏障那边等了很久了。"
