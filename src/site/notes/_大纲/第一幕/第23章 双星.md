# 第22章 双星

接下来的七天，白月霖每天清晨准时出现在训练场。星璃娅没有要求过，是她自己来的。

训练的内容很简单，反复练习三个动作：凝聚、释放、收敛。凝聚是将神力汇集到指尖；释放是将它投射出去，冻结一朵花、一片叶子或一颗琉玥从溪里捞上来的鹅卵石；收敛是将所有外溢的神力全部收回体内，不浪费一丝一毫。星璃娅对她说，真正的神力控制，关隘在于能收敛多少而非释放多少。一个神祇的成熟度，是用力量的安静程度来衡量的。

白月霖花了三天才做到收敛到不再在走路时把脚下的草冻成冰针。又花了两天做到释放时鹅卵石的冻结厚度不超过指甲盖。到第七天，她已经能控制神力在一片落叶的叶脉上写出"月"字，而不伤及叶肉。

"进步很快。"星璃娅坐在石柱的阴影里，手里翻着从黎敖书房顺来的《旧世界编年史》残卷。琉玥趴在她膝盖上打盹，尾巴一甩一甩。白月霖擦了一把额头上的汗，将那片写了自己名字的落叶小心地放进石像底座的小凹槽里。那里已经攒了七片叶子了。像是给石像献上的贡品。同时也是一种丈量：每天一片叶子，看看自己能走多远。

"星璃娅姐姐，你今天一直翻到书的最后一页然后停住了。上面写了什么？"

星璃娅没有回答，只是将残卷翻回前面某页，递给白月霖看。

那一页的纸质和其他页不一样，是某种兽皮硝制的，比纸更柔韧，也更抗岁月侵蚀。页面上有一幅手工绘制的星图。六颗恒星排成六芒星的阵型，每一颗与下一颗之间都有极细的银线相连。银线的交叉点上标注着一行古文字。白月霖不认识。但她认得那个阵型的形状，和冰窖里刻在冰壁上的封印阵一模一样。

"六星追日。"星璃娅说，"封印凤凰王的大阵。六颗恒星是锚点，银线是锚定链接。交叉点，这里，是阵核，也就是凤凰王被钉住的地方。"她的手指沿着一条银线缓缓划过，指甲在兽皮上留下浅痕。"但这个阵有一个漏洞：这六颗恒星都在缓慢移动。它们和所有的恒星一样，以极慢的速度在宇宙中移动。祈尔米修罗设阵的时候计算了它们的轨道，保证六条锚定链接的压力始终均等。但是——千年了。恒星已经移到了这个位置。"

她的手指点在星图右上角的一颗恒星上。那颗星比其他五颗明显偏了一小段距离。

"锚定力不再平衡。封印正在从一处边缘开始崩解。"

"崩解会怎么样？"

"两种可能。要么封印彻底失效，凤凰王出来，而且封印崩解瞬间释放的能量会把方圆数百里夷为焦土。要么，有人在封印完全崩解之前，主动从阵核内部反向突破。这样他不仅能提前出来，还能把封印阵残余的六颗恒星能量全部吸收。六颗恒星的能量。你知道那是多少吗？"

白月霖摇头。

"足够让他以接近完全体的形态降临这个世界，然后让熵余者的低语经由他覆盖到整个神寰。"星璃娅合上残卷，"所以他没有急着突破。他在等。等封印自然崩解到最后一道防线，然后一口吞掉所有残余能量。"

"迦尔姆说凤凰王在圣火里和他对话过。上周的事。"琉玥抬起头接话，她的声音带着刚睡醒的沙哑，但内容已经过了三遍逻辑推敲，"如果他能通过圣火投影，说明圣火在他和外部世界之间建立了一条神念通道。这种通道是双向的。理论上，我们可以反向追踪这条通道，锁定六星追日的时空坐标。"

"没错。"星璃娅将残卷搁在膝上，"有了坐标，就能直接传送到阵核外围，不需要翻遍整个神寰去找。但以我们目前的战斗力，正面迎战一个吸收了六颗恒星能量的堕落神祇，是送死。"她站起身，把还在打盹的琉玥从膝盖上轻轻挪开，然后向白月霖伸出一只手。

"下一步：学会一边控制你的锚定神力，一边同时承受来自外部的神力压制。从明天开始，我会用我的神力压制你。你要在我的压制下坚持释放和收敛，直到能在压制下自然呼吸为止。"

"还有，"白月霖握住她的手站起来，拍了拍裙摆上的草屑，动作里已经没有了七天前那种过分的小心翼翼，"告诉我怎么反向追踪那条圣火通道。"

星璃娅微微挑眉。"你打算自己去找凤凰王？"

"不。但我们得先知道他在哪。才能让他知道我们在找他。"

星璃娅看了她片刻。然后从残卷里抽出那张星图，放在白月霖手心里。

"这本书是黎敖的。星图也是他画的。他花了九百年来计算这六颗恒星的轨道偏移量。他自己没有神力去反向追踪，但他把所有的计算都做好了。只等一个能追踪的人。"

白月霖低头看星图。银线交织的阵型下，有一行极小的字，她一开始以为是标注，现在才看清那是黎敖的笔迹：

「艾瑟拉。九百三十七年零四个月。快好了。再等等。」

她将星图收进怀里，贴着心口。对那个在钟楼上守了千年时令的人，她还欠一句回复。

不过还不到时候。现在，她要先学会在别人压制她的神力的同时，站稳。
